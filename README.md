Data-driven procurement dashboard for vendor selection using scoring, analytics, and role-based management.

💡 Intelligent Vendor Selection & Procurement Optimizer

A data-driven procurement analytics dashboard that helps businesses evaluate and select the best vendors using a weighted scoring model based on price, delivery time, and ratings.

🚀 Features

🔐 User Authentication – Role-based login system (Admin/User)
📊 Vendor Ranking – Intelligent scoring algorithm for optimal selection
🧠 Decision Insights – Smart alerts and KPI metrics
📈 Interactive Dashboard – Visualizations using Plotly
🔍 Search & Filters – Dynamic filtering by category, location, rating
✏️ CRUD Operations – Add, edit, delete vendors (Admin access)
📤 Data Export – Download data in CSV and Excel formats
⭐ Favorites – Mark and track preferred vendors

How It Works
Vendor Data → Filtering → Scoring Algorithm → Ranking → Insights & Visualization

🛠️ Tech Stack:

Frontend: Streamlit
Backend: Python
Database: SQLite
Visualization: Plotly
Data Handling: Pandas

Use Case:

This system simulates a real-world procurement environment where organizations can analyze vendor performance and make informed decisions based on multiple criteria.

Project Summary:

This project demonstrates full-stack development skills by combining database management, authentication, analytics, and UI design into a single business-focused application.
